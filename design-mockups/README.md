# ESCEMI Website Design Mockups

This directory contains HTML mockups for 6 different artistic directions for the ESCEMI website redesign.

## How to View

Open any HTML file in a web browser to see the design mockup. Each file is self-contained with inline CSS.

## Available Directions

- **direction-3-neural-network.html** - Abstract tech with data visualization and animated nodes
- **direction-5-product-studio.html** - Apple-esque minimalism with extreme whitespace
- **direction-6-devops-horizon.html** - Dynamic gradients with animated mesh backgrounds
- **direction-7-brutalist-impact.html** - Bold brutalist design with thick borders
- **direction-8-magazine-editorial.html** - High-end editorial magazine layout
- **direction-9-neon-tech.html** - Futuristic cyberpunk with neon effects

## Branding

All designs respect ESCEMI branding:
- Primary color: #1c3144 (navy)
- Secondary color: #ecb807 (gold)
- Font: Source Sans Pro

## Positioning

All designs showcase the cv-coach positioning:
- Fractional CTO
- DevX Expert  
- Node.js & PHP Specialist (NestJS, Symfony, Laravel)
